The conversion of many of the buildings slated to be classrooms is not yet complete, so ad-hoc meeting spaces include hallways, outdoors, all corners of the Pomeroy Center (even the kitchen area!), and a nearby elementary school. One Pioneer remembers these earliest days fondly:

"[At the Pomeroy Center], there was only a partition between classes in some cases, so if you got bored with the lecture you were listening to, you could listen to the lecture from another professor's class nearby and learn something new."
